# MoveFit — EP4 Banco de Dados

Este repositório contém todos os scripts SQL utilizados na Experiência Prática 4 da disciplina de Banco de Dados.

## Arquivos incluídos

### 00_create_tables.sql
Contém o esquema do banco (DDL):
- Criação das tabelas
- Chaves primárias e estrangeiras
- Relacionamentos

### 01_insert.sql
Comandos INSERT para povoar:
- Planos
- Alunos
- Treinos
- Pagamentos
- Aulas
- Participação em aulas

### 02_select.sql
Consultas utilizando:
- JOIN
- WHERE
- ORDER BY
- LIMIT

### 03_update.sql
Três comandos UPDATE com condições.

### 04_delete.sql
Três comandos DELETE garantindo integridade.

## ✔️ Como executar
1. Abra o DB-Fiddle ou Workbench/PGAdmin.  
2. Cole o **00_create_tables.sql**.  
3. Cole o **01_insert.sql**.  
4. Execute as consultas nos arquivos seguintes.
